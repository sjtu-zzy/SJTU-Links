\## 🌟 项目简介



\*\*交我导 (SJTU Links)\*\* 是一款专为上海交通大学师生及校友设计的极速导航工具。

该项目采用纯前端架构，提供了极佳的响应速度与交互体验。通过系统化的分类与实时搜索，帮助用户从繁杂的校园网站与海量的公众号中，一键直达所需服务。



> ⚠️ \*\*免责声明\*\*：本项目的网站条目主要依据 \*\*上海交通大学官网「院系设置」和「机构设置」\*\* 整理校准，属于\*\*非官方开源项目\*\*。



\---



\## ✨ 核心特性



\- 🌐 \*\*多语言支持 (i18n)\*\*：完美支持 \*\*中文\*\* 与 \*\*English\*\* 一键无缝切换。

\- 🌓 \*\*智能深色模式\*\*：支持 \*\*浅色、深色、跟随系统\*\* 三种主题模式，夜间使用不刺眼。

\- 🔍 \*\*智能实时检索\*\*：支持按拼音、中文、英文、分类多维度模糊搜索，匹配关键词高亮（支持一键清空与重置筛选）。

\- 📑 \*\*灵活筛选与分页\*\*：

&#x20; - 支持 \*\*网站 (Website)\*\* 与 \*\*微信公众号 (WeChat)\*\* 分类过滤。

&#x20; - 动态标签卡（Chips）细分筛选（如：学院、部门、医院等）。

&#x20; - 自适应流式卡片，支持自定义每页显示条数（Pagination）。

\- 🐧 \*\*趣味交互小宠物「导导」\*\*：

&#x20; - 拥有高颜值 SVG 动态外观、会眨眼、眼睛随鼠标转动。

&#x20; - 支持\*\*自由拖拽与边缘贴边碰撞\*\*。

&#x20; - 丰富的互动台词气泡（如：“啾\~ 迷路了吗？”、“累了就摸摸我吧\~”）。

&#x20; - 如果觉得占地方，还支持一键隐藏与右下角魔法唤回。

\- 🤖 \*\*内置 AI 智能对话助手\*\*：

&#x20; - 侧边栏集成了流式 AI 助手聊天窗口。

&#x20; - 内置精美气泡对话框及上下文记忆能力，随时解答校园问题（需配置 API 服务连接）。

\- 📱 \*\*极致的移动端适配\*\*：针对手机、平板等窄屏设备进行了深度重构与触摸优化，支持手势拖拽宠物及滑块导航。

\## 🌐 部署至 GitHub Pages（免费静态托管）



通过 GitHub Pages，你可以轻松将本项目发布为在线网站，供所有人访问。



1\. 进入你 GitHub 的该项目仓库。

2\. 点击顶部的 \*\*Settings\*\* (设置)。

3\. 在左侧菜单栏中找到并点击 \*\*Pages\*\*。

4\. 在 \*\*Build and deployment\*\* 下的 \*\*Branch\*\* 区域：

\* 将分支选择为 `main` (或 `master`)。

\* 文件夹选择 `/ (root)`。





5\. 点击 \*\*Save\*\* 保存。

6\. 等待 1\~2 分钟，GitHub 会生成一个专属链接（例如：`https://你的用户名.github.io/你的仓库名/`），点击即可在线访问！
